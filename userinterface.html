User Interface (UI) Design focuses on anticipating what users might need to do and ensuring that the interface has elements that are easy to access, understand, and use to facilitate those actions.

This workshop will take a practical look at how to design an application with a/an attractive, simple to use, understandable, quick response front-end view which the user can interact with on all interface screens.
